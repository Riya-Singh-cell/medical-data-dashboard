🔗 **Live Demo:** https://medical-data-dashboard-h2ytjhyxawoug6z888ct83.streamlit.app/

---

## 📊 Overview
This project is an **interactive Healthcare Analytics Dashboard** built using **Streamlit, Pandas, and Plotly**.  
It helps analyze patient data, identify trends, and generate meaningful insights through visualizations.

---

## 🚀 Features

- 📌 **KPI Metrics**
  - Total Patients
  - Average Age
  - Average Blood Pressure
  - Cholesterol Levels
  - Most Common Disease  

- 🎯 **Interactive Filters**
  - Gender filter  
  - Disease filter  
  - Date range filter  

- 📈 **Visualizations**
  - Disease Distribution (Bar Chart)  
  - Gender Distribution (Pie Chart)  
  - Age Distribution (Histogram)  
  - Correlation Heatmap  
  - Top 5 Diseases  
  - Patient Outcome Analysis  

- 🧠 **AI-Style Insights**
  - Automatically generated key observations  

- 📋 **Data Table**
  - View full dataset dynamically  

- 📥 **Download Feature**
  - Export filtered data as CSV
